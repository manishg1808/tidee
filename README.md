# Tidee - Premium Dry Clean & Laundry Service Website

A modern, responsive website for Tidee, a premium dry clean and laundry service brand. Built with HTML, CSS (Tailwind), and JavaScript.

## 🌟 Features

- **Fully Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX** - Clean, professional design with smooth animations
- **SEO Optimized** - Proper meta tags, semantic HTML, and optimized content
- **Interactive Elements** - Booking forms, mobile navigation, testimonials slider
- **Custom Branding** - Brand logo, favicon, and professional color scheme
- **Multiple Pages** - Home, Services, Pricing, About, and Contact pages
- **Brand Identity** - Navy Blue (#0F2A44) and Light Cyan (#3cc9e8) color scheme with custom logo
- **Accessibility** - Screen reader friendly, keyboard navigation support

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local server for testing (optional but recommended)

### Running the Website

1. **Clone or download** the project files
2. **Navigate** to the project directory
3. **Start a local server** (recommended):
   ```bash
   # Using Python
   python -m http.server 8000

   # Using Node.js (if available)
   npx serve .

   # Or open directly in browser
   ```
4. **Open** `http://localhost:8000` in your browser
5. **Navigate** through the pages using the menu

### File Structure Details

- **HTML Files**: Clean, semantic HTML5 with proper headings hierarchy
- **CSS**: Tailwind CSS via CDN + custom styles in `style.css`
- **JavaScript**: Interactive functionality in `script.js`

## 🔧 Customization

### Colors
Edit the Tailwind color classes in HTML files:
- Primary: `text-[#0F2A44]` or `bg-[#0F2A44]`
- Secondary: `text-[#1FC8A9]` or `bg-[#1FC8A9]`

### Content
Update text content, images, and contact information directly in the HTML files.

### Functionality
Modify JavaScript in `js/script.js` for custom interactions.
